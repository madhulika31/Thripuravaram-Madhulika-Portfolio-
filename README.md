# 🌐 Personal Portfolio Website

A modern, responsive, and interactive portfolio website developed using HTML, CSS, and JavaScript. This project showcases my education, technical skills, projects, certifications, and contact information through a clean and professional user interface.

## ✨ Features

- Responsive design for all devices
- Modern and attractive UI
- Typing text animation
- Dark/Light mode toggle
- Smooth scrolling navigation
- Animated skill progress bars
- Education timeline
- Project showcase
- Certifications section
- Contact form
- Easy to customize

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🚀 Getting Started

1. Download or clone this repository.
2. Add your profile image as `profile.jpg`.
3. Add your resume as `resume.pdf`.
4. Open `index.html` in any web browser.
